# Travel Booking UI

A responsive and modern Travel Booking Website UI built using HTML and CSS.

This project focuses on creating an attractive travel landing page with clean design, responsive layouts, and smooth user experience.

## Features

- Responsive design
- Modern travel landing page
- Attractive hero section
- Destination cards
- Navigation bar
- Footer section
- Clean UI design
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3

## Project Structure

```bash
travel_booking_ui/
│
├── index.html
├── style.css
├── images/
└── README.md
```

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/travel_booking_ui.git
```

### 2. Open the project folder

```bash
cd travel_booking_ui
```

### 3. Run the project

Open `index.html` in your browser.
